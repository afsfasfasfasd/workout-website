Workout Plan

Pull 1 – Lat Focused

  • Weighted Pull-Up – 3 sets x 6 reps
  • Seated Cable Row – 3 sets x 10-12 reps
  • Cable Pullover (Kneeling) – 3 sets x 15-20 reps
  • Hammer Cheat Curl – 3 sets x 8-10 reps
  • Incline Dumbbell Curl – 2 sets x 12-15 reps

Push 1 – Chest Focused

  • Bench Press – 3 sets x 8 reps (72.5% 1RM)
  • Machine Shoulder Press – 3 sets x 12 reps
  • Dip – 3 sets x 12-15 reps (Deficit Pushups as replacement)
  • Skull Crusher (Eccentric-Accentuated) – 3 sets x 8-10 reps
  • Egyptian Lateral Raise – 3 sets x 12 + MYO on last
  • Cable Tricep Kickback – 3 sets x 20-30

Legs 1 – Quad Focused

  • Sample Pyramid Warm-up
  • Squats – 3 sets x 4 reps (80% 1RM)
  • RDL – 3 sets x 10 reps
  • Single Leg Press – 3 sets x 15 reps
  • Eccentric-Accentuated Leg Extension – 3 sets x 10-12 reps
  • Seated Leg Curl – 3 sets x 10-12 reps
  • Standing Calf Raise – 3 sets x 10-12 reps
  • Decline Crunches – 2 sets x 10-12 reps
  • Long-Lever Planks – 2 sets x 30s

Pull 2 – Mid-Back & Rear Delt Focused

  • OMNI-Grip Lat Pulldown – 3 sets x 10-12
  • Chest-Supported Row – 3 sets x 10-12
  • Rope Face Pull – 3 sets x 15-20
  • Incline Dumbbell Shrug – 3 sets x 15-20 (optional)
  • Reverse Pec Deck – 2 sets x 15 + 10-15
  • Pronated/Supinated Curl – 3 sets x 10/10

Push 2 – Delt Focused

  • Overhead Press – 4 sets x 4 reps (80% 1RM)
  • Close-Grip Bench Press – 3 sets x 10 reps
  • Cable Crossover – 3 sets x 10-12 (+ drop set last)
  • Overhead Tricep Extension – 3 sets x 10-12 reps
  • Lateral Raise 21's – 3 sets (7/7/7)
  • Neck Flexion/Extension – 3 sets x 10-12

Legs 2 – Posterior Chain Focused

  • Deadlift – 3 sets x 3 reps (80-85% 1RM)
  • Hack Squat – 3 sets x 10-12 reps (Goblet Squats alt.)
  • Single-Leg Hip Thrust – 2 sets x 15 reps
  • Nordic Ham Curl – Unknown reps/sets
  • Prisoner Back Extension – Unknown reps/sets
  • Single-Leg Calf Raise – 3 sets x 8-10 reps
  • Weighted L-Sit Hold – 3 sets to failure
